## Instrucciones de Ejecución Local
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Configurar el archivo `.env` con las credenciales de Supabase y GitHub Auth.
4. Ejecutar `npx prisma generate` y `npx prisma db push`.
5. Iniciar con `npm run dev`.

La documentación técnica se encuentra disponible en la ruta: `/docs` (o `/api/docs`).
